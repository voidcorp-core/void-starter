# Phase C: Web App Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `apps/web` as a fully functional Next.js 16 application that consumes the Phase A + B packages and demonstrates the canonical patterns (sign-in/sign-up/reset-password/verify-email pages, protected dashboard, admin-only page, home page, canonical component examples). Add Vitest at app level and Playwright with end-to-end auth tests.

**Architecture:** `apps/web` is the only app at J0. It demonstrates how a Void Factory MVP consumes `@void/core`, `@void/auth`, `@void/db`, `@void/ui`. Server Actions live in `apps/web/src/actions/` and use `defineAction` from `@void/auth`. The `instrumentation.ts` is wired with stubs ready to receive opt-in modules in Phase D. Security headers from `@void/core` are wired in `next.config.ts`. Auth pages use Better-Auth's React client (`useSession`, `signIn`, `signOut` from `@void/auth/client`).

**Tech Stack added in this phase:** Next.js 16.2, React 19.2, @playwright/test, jsdom, @testing-library/react.

**Reference:** `context.md` Architecture principles + Topology, `starter-plan.md` Steps 7-9, `docs/DECISIONS.md` entries 03 (.actions placement), 04 (build-time module activation), 08 (service layout).

**Pre-conditions:**
- `git tag phase-b-complete` exists on origin
- `bun run test` passes all unit tests across `@void/core`, `@void/db`, `@void/auth`, `@void/ui`
- Local Postgres is running (`bun run db:up`) and migration applied

```ts
'use server';

import { redirect } from 'next/navigation';
import { z } from 'zod';
import { defineAction, signIn, signOut as signOutCore } from '@void/auth';

export const signInWithEmailAction = defineAction({
  schema: z.object({
    email: z.string().email(),
    password: z.string().min(8),
  }),
  auth: 'public',
  handler: async (input) => {
    const result = await signIn.email({ body: input });
    return { ok: true, result };
  },
});

export const signUpWithEmailAction = defineAction({
  schema: z.object({
    email: z.string().email(),
    password: z.string().min(8),
    name: z.string().min(1),
  }),
  auth: 'public',
  handler: async (input) => {
    // Use Better-Auth's signUp through the client; verify the actual API
    const { authClient } = await import('@void/auth/client');
    const result = await authClient.signUp.email(input);
    return { ok: true, result };
  },
});

export async function signOutAction() {
  await signOutCore();
  redirect('/');
}

export const requestMagicLinkAction = defineAction({
  schema: z.object({ email: z.string().email() }),
  auth: 'public',
  handler: async (input) => {
    await signIn.magicLink({ body: input });
    return { ok: true };
  },
});
```

### Task C10: Sign-in page

**Files:** Create `apps/web/src/app/(auth)/sign-in/page.tsx`

- [ ] **Read React 19 useActionState docs**: `https://react.dev/reference/react/useActionState`

- [ ] **Create the sign-in page** (client component for form state)

```tsx
'use client';

import { useActionState } from 'react';
import Link from 'next/link';
import { Button, Card, CardBody, CardHeader, Input, Label } from '@void/ui';
import { authClient } from '@void/auth/client';
import { signInWithEmailAction } from '@/actions/auth.actions';

type FormState = { error: string | null };

async function submit(_prev: FormState, formData: FormData): Promise<FormState> {
  const email = String(formData.get('email') ?? '');
  const password = String(formData.get('password') ?? '');
  try {
    await signInWithEmailAction({ email, password });
    window.location.href = '/dashboard';
    return { error: null };
  } catch (err) {
    return { error: err instanceof Error ? err.message : 'Sign in failed' };
  }
}

export default function SignInPage() {
  const [state, action, pending] = useActionState(submit, { error: null });

  return (
    <main className="mx-auto max-w-sm px-6 py-16">
      <Card>
        <CardHeader>
          <h1 className="text-2xl font-semibold">Sign in</h1>
        </CardHeader>
        <CardBody>
          <form action={action} className="space-y-4">
            <div className="space-y-2">
              <Label htmlFor="email">Email</Label>
              <Input id="email" name="email" type="email" required autoComplete="email" />
            </div>
            <div className="space-y-2">
              <Label htmlFor="password">Password</Label>
              <Input id="password" name="password" type="password" required autoComplete="current-password" />
            </div>
            {state.error && <p className="text-sm text-destructive">{state.error}</p>}
            <Button type="submit" disabled={pending} className="w-full">
              {pending ? 'Signing in...' : 'Sign in'}
            </Button>
          </form>
          <div className="mt-4 space-y-3">
            <Button
              variant="secondary"
              className="w-full"
              onClick={() => authClient.signIn.social({ provider: 'google', callbackURL: '/dashboard' })}
            >
              Continue with Google
            </Button>
            <p className="text-sm text-muted-foreground text-center">
              No account? <Link href="/sign-up" className="underline">Sign up</Link>
            </p>
          </div>
        </CardBody>
      </Card>
    </main>
  );
}
```
